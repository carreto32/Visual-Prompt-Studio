PROMP 1 MEDIUM A
Please create a graphite pencil drawing that focuses on value, texture, unity. Use graphite pencils doing blending and stippling to build the shading and textures. The drawing should show a skateboard leaning against a wall with really strong low lighting, creating dark shadows and light areas that make the skateboard stand out. 

PROMPT 2 MEDIUM B 
Please create a woodblock print that focuses on value, texture, and unity. Use wood carving tools to cut lines and shapes into the wood block, then print it with ink to create the image. The print should show a skateboard leaning against a wall with really strong low lighting, creating dark shadows and light areas so the skateboard stands out.
